Inspection Web App

ระบบรายงานผลการตรวจสอบหม้อแปลงไฟฟ้า

สำหรับบันทึกผลการตรวจสอบหน้างาน พร้อมหมายเลขหม้อแปลง พิกัดตำแหน่ง และรูปถ่ายจริง โดยส่งข้อมูลไปยัง Google Apps Script API เพื่อบันทึกลง Google Sheet และ Google Drive

รองรับข้อมูล
เลขหม้อแปลง / ผลตรวจ / พิกัด / รูปถ่าย
การเชื่อมต่อ
Google Apps Script API

สิ่งที่ต้องเตรียม

  • URL ของ Web App จาก Google Apps Script
  • สคริปต์ฝั่งหลังบ้านที่รองรับการรับข้อมูลแบบ JSON
  • สิทธิ์การเข้าถึงตำแหน่ง (Location)
หมายเหตุ: ไฟล์นี้เป็นหน้าเว็บไฟล์เดียว สามารถนำไปวางในระบบที่เรียกใช้ API จาก Apps Script ได้ทันที

บันทึกผลการตรวจสอบ

กรอกข้อมูลให้ครบถ้วนก่อนกดส่งรายงาน

หากต้องการล็อกค่า URL ไว้ถาวร สามารถกำหนดไว้ในตัวแปร DEFAULT_SCRIPT_URL ด้านล่างของไฟล์นี้

ระบบจะส่งข้อมูลเป็น JSON ไปยัง Apps Script โดยแนบรูปเป็น Base64 พร้อมชื่อไฟล์และชนิดไฟล์
